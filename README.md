# What's this?
My personal learning notes, a bit of everything: tech, music, food and whatever else I find interesting and can't stick in my head.
I write them for myself, but hey, maybe they'll save someone else a headache too.

# Website

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Deployment

This site is deployed to GitHub Pages automatically via GitHub Actions.
Every push to the `main` branch triggers the workflow defined in `.github/workflows/deploy.yml`, which builds the site and publishes it.
